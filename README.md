# Crossfriends - 기독교인 소셜 네트워크

## 프로젝트 개요
- **이름**: Crossfriends
- **목표**: 기독교인들만을 위한 소셜 네트워크 플랫폼
- **주요 기능**:
  - 게시물 작성 및 공유 (성경 구절 첨부 가능)
  - 댓글 및 좋아요 기능
  - 기도 제목 작성 및 응답
  - 사용자 프로필 관리
  - **관리자 패널** (회원/게시물 관리, 통계)
  - 역할 기반 접근 제어 (admin, moderator, user)
  - **💰 수익 계산기** - 회원 가입자 별 예상 수익 계산 및 시각화

## 🎨 색상 시스템 (7 Colors)

CROSSfriends는 일관된 디자인을 위해 **7가지 색상 체계**를 사용합니다.

### 색상 팔레트

| 색상 | 용도 | HEX 코드 | CSS 변수 |
|------|------|---------|---------|
| **Primary** (파란색) | 주요 버튼, 링크, 브랜드 | `#3B82F6` | `--color-primary` |
| **Accent** (빨간색) | 관리자 기능, 삭제, 경고 | `#DC2626` | `--color-accent` |
| **Success** (초록색) | 성공 메시지, 완료 상태 | `#16A34A` | `--color-success` |
| **Warning** (노란색) | 경고 메시지, 주의사항 | `#CA8A04` | `--color-warning` |
| **Info** (청록색) | 정보 메시지, 팁 | `#0891B2` | `--color-info` |
| **Neutral** (회색) | 텍스트, 배경, 테두리 | `#6B7280` | `--color-neutral` |

### 색상 사용 예시

- **Primary**: 로그인 버튼, 게시하기 버튼, 링크
- **Accent**: 관리자 패널, 삭제 버튼, 십자가 아이콘
- **Success**: 회원가입 완료 메시지
- **Warning**: 관리자 왕관 배지 (황금)
- **Info**: 정보 알림, 도움말
- **Neutral**: 일반 텍스트, 카드 배경

### CSS 변수 사용

```css
/* 색상 시스템 사용 예 */
.primary-button {
    background-color: var(--color-primary);
}

.accent-text {
    color: var(--color-accent);
}
```

- **언락 조건**: 종합점수 500점 달성
- **종합점수 계산**: 성경 점수 + 기도 점수 + 활동 점수
- **리워드 내용**: 오늘의 설교 영상 시청 권한
  - 조용기 목사님의 "낙망하고 불안해하지 말라" (시편 42:5)
  - YouTube 영상 재생 가능
  - 90% 이상 시청 시 추가 100점 획득

### 리워드 화면 구성
- **잠금 상태 (< 500점)**
  - 👑 황금 왕관 아이콘 (애니메이션)
  - 성경 말씀 표시

  - 현재 점수 및 진행 바
  - 목표까지 남은 점수 표시
  - 잠금 버튼 (비활성화)

- **언락 가능 (= 500점)**
  - 언락 버튼 활성화 (펄스 애니메이션)
  - 클릭 시 설교 영상 공개
  - 🎉 축하 메시지

- **언락 완료 (> 500점)**
  - 🏆 트로피 및 축하 배너
  - 설교 영상 전체 공개
  - 시청 진행도 추적
  - 완료 시 보너스 점수

### 점수 획득 방법
| 활동 | 점수 | 카테고리 |
|------|------|----------|
| 포스팅 작성 | 10점 | 카테고리별 (성경/기도/활동) |
| 댓글 작성 | 5점 | 활동 점수 |
| 좋아요 클릭 | 1점 | 카테고리별 |
| 기도 클릭 | 20점 | 기도 점수 |
| 말씀 타이핑 (100% 정확) | 100점 | 성경 점수 |
| 설교 시청 (90%+) | 100점 | 성경 점수 |

### 500점 달성 팁
- **빠른 달성 방법**:
  1. 말씀 타이핑 5회 완벽 입력 (500점)
  2. 포스팅 25개 + 댓글 50개 작성 (500점)
  3. 기도 포스팅 5개 + 기도 클릭 20회 (450점) + 댓글 10개 (50점)

## 📊 데이터 모델

### 사용자 (Users)
- 이메일, 이름, 소개, 아바타
- 소속 교회, 교단, 지역 정보
- **결혼 유무** (marital_status): 미혼(single), 기혼(married), 기타(other)
- **🔒 프라이버시 설정** (privacy_settings): 카테고리별 비공개 설정
  - `basic_info`: 기본 정보 (이메일, 성별, 결혼 유무, 주소, 전화번호)
  - `church_info`: 교회 정보 (교회, 담임목사, 교단, 지역, 직분)
  - `education_info`: 학력 정보 (초/중/고/대학, 전공, 석박사)
  - `career_info`: 직업 정보 (경력 사항)
  - `scores`: 점수 정보 (성경/기도/활동 점수)
- **점수 시스템**:
  - `scripture_score`: 성경 구절 타이핑 & 설교 듣기 점수
  - `prayer_score`: 기도 제목 작성 및 기도하기 점수
  - `activity_score`: 게시물, 댓글 작성 등 활동 점수

### 게시물 (Posts)
- 내용, 이미지, 성경 구절 참조
- 작성자 정보, 좋아요/댓글 수

### 댓글 (Comments)
- 게시물에 대한 댓글
- 작성자 정보, 작성 시간

### 좋아요 (Likes)
- 게시물 좋아요 기능
- 중복 방지 (UNIQUE 제약)

### 기도 제목 (Prayer Requests)
- 제목, 내용, 익명 여부
- 상태 (진행중/응답됨/종료)
- 기도 응답 수 추적

### 기도 응답 (Prayer Responses)
- 기도 제목에 대한 응답
- 격려와 함께 기도

### 친구 관계 (Friendships)
- 친구 요청 및 수락/거절
- 상태 관리 (대기/수락/거절)

## ✅ 최근 구현된 기능

### 🔐 관리자 시스템 (Admin Panel)
- **역할 기반 접근 제어 (RBAC)**
  - user: 일반 사용자
  - moderator: 운영자 (향후 확장 가능)
  - admin: 최고 관리자
  
- **첫 가입자 자동 관리자**: 첫 번째 회원가입자가 자동으로 admin 권한 획득

- **관리자 대시보드** (`/admin`)
  - 실시간 통계 카드 (회원, 게시물, 댓글, 기도 제목)
  - 회원 관리: 목록 조회, 역할 변경, 회원 삭제
  - 게시물 관리: 목록 조회, 게시물 삭제
  - 깔끔한 테이블 UI with TailwindCSS
  
- **🌱 개연성 시뮬레이션 (NEW!)** ⭐
  - **자연스러운 커뮤니티 활동 시뮬레이션**
    - 신규 회원 가입 (3-7일마다 1-3명)
    - 일일 포스팅 생성 (하루 2-8개)
    - 댓글 자동 생성 (포스팅의 30%)
    - 좋아요/반응 생성 (포스팅의 40%)
    - 기도 클릭 생성 (기도 포스팅의 50%)
  - **시간대별 분산**
    - 포스팅: 하루 중 랜덤 시간
    - 댓글: 포스팅 후 12시간 이내
    - 좋아요: 포스팅 후 24시간 이내
  - **점수 자동 반영**
    - 포스팅: 10점 (카테고리별 차등)
    - 댓글: 5점 (활동 점수)
    - 좋아요: 1점 (카테고리별 차등)
    - 기도 클릭: 20점 (기도 점수)
  - **사용법**: Admin 페이지 → "🌱 개연성 시뮬레이션" 버튼 클릭 → 일수 입력 (1-365일)
  - **예상 소요 시간**: 일수 ÷ 10 분 (14일 = 약 1.4분)
  
- **관리자 API**
  - `GET /api/admin/stats` - 전체 통계
  - `GET /api/admin/users` - 회원 목록 + 활동 통계
  - `PUT /api/admin/users/:id/role` - 역할 변경
  - `DELETE /api/admin/users/:id` - 회원 삭제
  - `GET /api/admin/posts` - 게시물 목록
  - `DELETE /api/admin/posts/:id` - 게시물 삭제
  - `POST /api/admin/simulate-time-pass` - 개연성 시뮬레이션 (NEW!)
  - Middleware를 통한 권한 검증 (X-Admin-ID 헤더)

### 🔒 프라이버시 시스템 (NEW!)
- **카테고리별 비공개 설정**
  - 사용자가 5가지 카테고리를 독립적으로 공개/비공개 설정 가능
  - 비공개 설정 시 해당 카테고리의 **모든 정보가 완전히 숨김 처리**
  
- **비공개 카테고리 목록**
  1. **기본 정보**: 이메일, 성별, 결혼 유무, 주소, 전화번호
  2. **교회 정보**: 교회, 담임목사, 교단, 지역, 직분
  3. **학력 정보**: 초/중/고/대학, 전공, 석박사 학위
  4. **직업 정보**: 경력 사항 (회사, 직책, 기간)
  5. **점수 정보**: 성경 점수, 기도 점수, 활동 점수

- **비공개 처리 방식**
  - API 응답: 해당 필드 값이 `null`로 반환
  - HTML 프로필: "🔒 [카테고리명]이 비공개로 설정되어 있습니다" 메시지 표시
  - **입력 여부 무관**: 정보가 입력되어 있어도 비공개면 다른 사용자는 볼 수 없음
  
- **본인 프로필 조회**
  - 본인이 자신의 프로필을 조회할 때는 모든 정보 표시
  - `current_user_id` 파라미터로 본인 여부 확인

- **API 엔드포인트**
  - `GET /api/users/:id?current_user_id={id}` - 프라이버시 필터 적용된 사용자 조회
  - `PUT /api/users/:id` - 프라이버시 설정 업데이트 (privacy_settings 필드)

1. 메인 페이지 우측 상단의 **"수익 계산기"** 버튼 클릭
2. 또는 직접 `/revenue` URL로 접속

#### 수익 계산 방법
1. **가입자 수 입력**
   - 예상 회원 수를 입력 (100명 단위 권장)
   - 빠른 선택 버튼 활용 가능 (500, 1000, 5000, 10000, 50000명)

2. **"수익 계산하기"** 버튼 클릭

3. **결과 확인**
   - 📊 요약 카드: 총 회원, 유료 회원, 월/연 수익
   - 📈 차트: 수익 구성, 플랜별 분포, 예상 추이
   - 📋 상세 분석: 구독료/광고 수익 세부 내역

#### 가정 사항
- **유료 전환율**: 30% (무료 회원 중 30%가 유료로 전환)
- **플랜 분포**: 
  - 베이직(₩9,900): 60%
  - 프로(₩29,900): 30%
  - 엔터프라이즈(₩99,900): 10%
- **추가 수익**: 회원당 월 ₩100 (광고, 후원 등)

1. 우측 상단의 "로그인" 버튼 클릭
2. 이메일 주소 입력 (등록된 이메일)
3. "로그인" 버튼 클릭

**테스트 계정:**
- **관리자**: `holofa518@gmail.com` (관리자 - **회원가입 없이 바로 로그인 가능**)
  - 이 이메일은 회원가입 없이도 자동으로 관리자 계정이 생성됩니다
  - 로그인 시 자동으로 admin 권한이 부여됩니다

#### 관리자 패널 접속
**관리자 계정으로 로그인 후:**
1. 우측 상단에 빨간색 "관리자" 버튼 표시됨
2. "관리자" 버튼 클릭 또는 `/admin` URL 직접 접속
3. 관리자 대시보드에서 다음 작업 가능:
   - **통계 확인**: 회원, 게시물, 댓글, 기도 제목 수
   - **회원 관리**: 역할 변경 (user ↔ moderator ↔ admin), 회원 삭제
   - **게시물 관리**: 부적절한 게시물 삭제
4. 일반 사용자는 "관리자" 버튼이 보이지 않으며, `/admin` 접속 시 홈으로 리다이렉트

**관리자 기능:**
- ✅ 실시간 통계 모니터링
- ✅ 회원 역할 변경
- ✅ 회원 삭제 (본인 제외)
- ✅ 게시물 삭제
- ✅ 회원별 활동 통계 (게시물, 댓글, 기도 수)

#### 회원가입 방법
1. 우측 상단의 "회원가입" 버튼 클릭
2. 모든 기본 정보 입력:
   - 이름, 성별, 이메일
   - 프로필 사진 (선택, 최대 5MB)
   - 교단 선택
   - 소속 교회, 담임목사
   - 교회 위치 (도→시 2단계 선택)
   - 교회 직분
   - **결혼 유무** (미혼, 기혼, 기타 - 선택사항)
3. 신앙 고백 10가지 질문에 답변
4. "회원가입 완료" 버튼 클릭
5. 자동으로 로그인됩니다

**첫 가입자 특전:** 첫 번째 회원가입자는 자동으로 관리자 권한을 받습니다! 🎉

#### 프로필 수정 방법
1. 로그인 후 우측 상단의 프로필 클릭
2. 수정할 정보 입력:
   - 이름, 성별
   - 교회, 담임목사, 직분
   - **결혼 유무** (미혼, 기혼, 기타)
3. "저장하기" 버튼 클릭

**주의사항:**
- 이메일은 고유 식별자로 변경할 수 없습니다
- 프로필 사진은 JPG, PNG만 가능 (최대 5MB)
- 회원가입 시 이미 등록된 이메일은 사용할 수 없습니다

### 📝 게시물 작성
1. 상단의 텍스트 입력 창에 내용을 작성합니다
2. (선택) 성경 구절 참조를 입력합니다 (예: 시편 23:1)
3. "게시하기" 버튼을 클릭합니다

### 댓글 작성
1. 게시물의 댓글 아이콘을 클릭합니다
2. 댓글 입력창에 내용을 작성합니다
3. 전송 버튼을 클릭합니다

### 좋아요
- 게시물의 하트 아이콘을 클릭하면 좋아요/취소가 됩니다
- 빨간 하트는 좋아요 상태를 나타냅니다

### 기도 제목 작성
1. 우측 상단의 "기도 요청" 버튼을 클릭합니다
2. 제목과 내용을 작성합니다
3. (선택) 익명으로 게시할 수 있습니다
4. "기도 요청 올리기" 버튼을 클릭합니다

### 사용자 전환 (테스트용)
- 우측 상단의 드롭다운에서 다른 사용자를 선택할 수 있습니다
- 각 사용자별로 다른 좋아요 상태가 유지됩니다

## 🛠️ 개발 가이드

### 로컬 개발 환경 설정

```bash
# 의존성 설치
cd /home/user/webapp
npm install

# 데이터베이스 마이그레이션
npm run db:migrate:local

# 시드 데이터 추가
npm run db:seed

# 프로젝트 빌드
npm run build

# 개발 서버 시작
pm2 start ecosystem.config.cjs

# 서버 상태 확인
pm2 list

# 로그 확인
pm2 logs crossfriends --nostream

# 서버 재시작
fuser -k 3000/tcp 2>/dev/null || true
pm2 restart crossfriends

# 데이터베이스 리셋 (개발용)
npm run db:reset
```

```bash
# 1. Cloudflare API 키 설정 (최초 1회)
# Deploy 탭에서 API 키를 설정하세요

# 2. 프로덕션 데이터베이스 생성 (최초 1회)

# 3. 프로덕션 마이그레이션
npm run db:migrate:prod

# 4. Pages 프로젝트 생성 (최초 1회)
npx wrangler pages project create crossfriends --production-branch main

# 5. 배포
npm run deploy:prod

```
