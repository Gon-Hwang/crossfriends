### 4️⃣ 데이터베이스 점수 확인 테스트

#### 테스트 4-1: DB 직접 조회
```sql
SELECT id, name, scripture_score, prayer_score, activity_score,
       (scripture_score + prayer_score + activity_score) as total_score
FROM users
WHERE id = 43;
```

**예상 최종 점수:**
- 성경 점수: 11점 (포스팅 10 + 반응 1)
- 기도 점수: 30점 (포스팅 10 + 반응 20)
- 활동 점수: 70점 (포스팅 50 + 반응 5 + 댓글 15)
- **종합 점수: 111점**
